## Installation & Setup

> Server side

###### prerequisites

```bash
$ git clone 
$ cd poking-app
$ cd server
$ npm install 
$ npm run dev # for development
$ npm run prod # for production
```

###### ⚠️ Do not forget to add your own `.env` file

##### change env.example file name to config.env and add your own values.
